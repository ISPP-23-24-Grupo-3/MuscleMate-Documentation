## Crear evento

Dado un propietario de gimnasio logueado en la plataforma

Al pinchar en `Eventos` el cliente debería ver un formulario donde pueda
crear clases para sus gimnasios.

El formulario debe tener los siguientes campos:

- Un campo llamado `Titulo del evento` (CAMBIAME Nos quedamos con eventos o clases?) informando de la clase brevemente
- Un campo llamado `Descripcion` donde pueda describir la clase en profundidad
- Un campo `Aforo` (--ref) donde pueda indicar las personas que van a asistir a la clase
- Un campo `Asistentes` (CAMBIAME esto porque hace falta especificarlo si los clientes se apuntan)
- Un campo `Monitor` donde pueda escribir la persona que va a impartir la clase
- Un campo `Fecha del evento` (CAMBIAME Es una fecha falta la hora de comienzo) donde el usuario pueda especificar el dia y la hora
  de la clase
- Duracion (minutos) (CAMBIAME No es mejor fecha y hora de fin, la duracion es un atributo calculado)
- Intensidad del evento donde el usuario pueda especificar el grado de intensidad de la
  clase
- Por ultimo un campo `Gimnasio` donde el usuario pueda especificar en que gimansio
  se va a impartir la clase

## Listar eventos

Al pulsar los `Eventos` debe aparecer un listado de clases agrupadas por gimnasio:

Por ejemplo

BasicFit Camas:

- Clase de pilates
- Clase de yoga

NewFitnes Tomares:

- Clases de Spinning
- Clases de bodypump

Si el propietario no tiene gimnasios registrados se le sugerirá al
propietario crear uno mediante un enlace.

## Eliminar eventos

- Si un propietario elimina una clase, se mostrará un mensaje de confimación
  si tiene exito
- Si se elimina una clase se eliminaran todas las personas que se han inscrito
  a esa clase
- Si el propietario intenta eliminar una clase de otro propietario, se le
  mostrará una pantalla de error diciendo que no está autorizado para realizar
  esa acción
