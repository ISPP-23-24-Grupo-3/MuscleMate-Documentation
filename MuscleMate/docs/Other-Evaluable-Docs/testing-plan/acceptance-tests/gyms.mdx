### Visualizar gimnasios

Se monstrará una lista de los gimnasios que el propietario tiene registrado

- Si el propietario no tiene gimnasios registrados, se tendrá que ve un mensaje
  incitando al usuario a utiliza las característica de Crear Nuevo Gimnasio

### Eliminar gimnasio

- Si el propietario hace click en `Eliminar gimnasio` se eliminarán todos los
  datos relacionados con el gimansio (las máquinas del gimnasio, los usuarios del
  gimnasio, las valoraciones de las máquinas del gimnasio).

Al ser una acción muy destructiva, se mostrará un diálogo de confirmación de
borrado de gimnasio.

- Se le mostrará un mensaje al propietario confirmando el borrado del gimnasio

- Un propietario no puede borrar el gimnasio de otro propietario.
