- Prueba hacer una peticion al backend con registrando al usuario en otro gimansio de otro propietario

### Visualizar usuarios

Se mostrará un listado de usuarios agrupados por gimnasios
donde prodra ver el nombre del usuario y el estado de su matricula

### Eliminar usuario

- Si el propietario decide borrar eliminar a un usuario se borraran todos los datos
  de sus rutinas y de sus ejercicios, asi como sus tickets y sus valoraciones
  Al ser una acción muy destructiva, se mostrará un diálogo de confirmación de
  borrado de la cuenta del usuario.

- Se le mostrará un mensaje al propietario confirmando el borrado del usuario

- Un propietario no puede eliminar el usuario de otro gimnasios
