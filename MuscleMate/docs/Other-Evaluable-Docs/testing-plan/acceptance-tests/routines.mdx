## Requisitos previos

- Toda la interfaz del caso de uso tiene que estar en español
- Los mensajes de error tienen que estar en español

## Crear rutina

Dado un cliente de gimnasio logueado en la plataforma

Al pinchar en `Rutinas` el cliente debería ver un botón para `Añadir sus rutinas`.

El formulario debe tener los siguientes campos:

- El formulario debe contener un campo `Nombre de la rutina` para que el usuario
  pueda identificarla.

- Un usuario no puede crear una rutina y asociarla a otro usuario

## Listar rutinas

Al pulsar en `Rutinas` dentro de la barra de navegación debe aparecer un listado de rutinas,
pudiendo hacer varias acciones como:

- Ver los detalles de una rutina
- Eliminar una rutina
- Editar una rutina

## Eliminar rutina

- Si un propietario elimina una rutina, se mostrará un mensaje de confimación
  si tiene exito
- Si se elimina una rutina se eliminaran todas los ejercicios asociados a la rutina
- Si el propietario intenta eliminar una rutina de otro propietario, se le
  mostrará una pantalla de error diciendo que no está autorizado para realizar
  esa acción
