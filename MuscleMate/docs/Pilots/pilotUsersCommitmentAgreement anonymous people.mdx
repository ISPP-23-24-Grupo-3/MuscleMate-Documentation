# Acuerdo de Compromiso Mutuo para Ayudar al Grupo 3 de ISPP

Este documento representa un acuerdo de compromiso mutuo entre las personas anónimas interesadas en ayudar al Grupo de Trabajo 3 de ISPP, con el fin de establecer las bases de colaboración y apoyo mutuo en las pruebas de sus respectivas aplicaciones. El cumplimiento de los términos establecidos en este acuerdo es crucial para asegurar el progreso y la calidad de los proyectos en desarrollo por el Grupo 3.

1. **Compromisos del Grupo 3 y las Personas Anónimas:**
   - **Despliegue de Aplicaciones:** El Grupo 3 se compromete a proporcionar a las personas anónimas un enlace desplegado con la aplicación cada lunes antes de las 15:00 horas. Este despliegue debe estar en un estado estable, listo para ser sometido a pruebas.
   - **Facilitación de Formulario de Feedback:** Cada viernes, el Grupo 3 se compromete a proporcionar a las personas anónimas un formulario para que respondan preguntas sobre la aplicación y proporcionen su feedback. Este formulario debe ser devuelto completado por las personas anónimas antes del domingo.
   - **Recepción del Formulario de Feedback:** Las personas anónimas se comprometen a devolver el formulario completo con sus respuestas y feedback antes del domingo de cada semana.

2. **Entrega de Tests Realizados:**
   - El Grupo 3 se compromete a entregar a las personas anónimas un mínimo de 8 tests realizados sobre la aplicación antes de las 15:00 del lunes, día siguiente al despliegue. Los resultados de estos tests deben incluir:
     - Resultados obtenidos.
     - Recomendaciones o sugerencias de mejora, si las hubiera.

3. **Comunicación y Resolución de Problemas:**
   - Para cualquier duda, aclaración o problema surgido durante el proceso de pruebas, las personas anónimas y el Grupo 3 se comprometen a comunicarse de manera abierta y constructiva, buscando siempre la resolución de problemas de manera colaborativa.

4. **Incumplimiento:**
   - El incumplimiento de forma reiterada por una de las partes habilita a la otra a abandonar la situación de acuerdo.

5. **Modificaciones al Acuerdo:**
   - Cualquier modificación a este acuerdo debe ser discutida y acordada por ambas partes. Las modificaciones solo serán efectivas una vez que sean documentadas por escrito y firmadas por representantes autorizados de ambas partes.
